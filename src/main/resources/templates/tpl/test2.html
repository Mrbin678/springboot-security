测试tpl中使用spring bean:
格式化时间函数：
\{\{ dream.use("fmt").format(d.date, "yyyy-MM-dd HH:mm:ss"); \}\}
\{\{ dream.use("fmt").format(d.localDateTime, "yyyy-MM-dd HH:mm:ss"); \}\}
输出： {{# var fmt = dream.use("fmt"); }} {{ fmt.format(d.date, 'yyyy-MM-dd HH:mm:ss') }}
{{ fmt.format(d.localDateTime, 'yyyy-MM-dd HH:mm:ss') }}
格式化数字：
注意:`#`在tpl中是特殊字符
\{\{# var _num = dream.use("fmt").format(d.num, "#.00"); \}\} \{\{ _num \}\}
输出： {{# var _num = fmt.format(100.999999,"#.00") }} {{ _num }}
