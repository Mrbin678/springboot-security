版本更新说明：

v2.0.3 2018-10-27

1. 使用 lutool 2.0.3 版本，针对Spring boot 2.x

2. 升级到 Spring boot 2.x、mybatis plus 3.x

v1.0.4 2018-07-21

1. 添加 druid 数据库密码加密

2. lutool 添加 ontest 环境

3. 优化lutool Fegin 和 RestTemplate日志配置

4. 添加 war 包支持

v1.0.3 2018-06-03

1. lutool 添加cloud模块，添加Fegin自动fallback功能(再也不用写一堆fallback)、集成ID生成服务

2. lutool 代码优化，集成jsr305和findBug静态代码检查

3. 添加如梦cloud Api接口和Service服务在线代码生成

4. lutool dream-security添加Druid可执行sql打印和数据库密码加密

v1.0.2 2018-05-20

1. spring boot 升级到 1.5.13.RELEASE

2. swagger 支持 Authorization 授权

3. 优化Aop事物只有在添加 spring-tx 之后启用

4. 添加针对 Boolean、Integer、Long的状态模式格式化 @StatusFormat

5. 验证码默认改为关闭

6. 修改 @ApiVersionMapping 只能注解到类上面，避免误用照成混乱，组合@Validated，方便接口参数校验

7. 增强校验，支持方法(@Validated @NotBlank String test) 也可用于服务层

8. 添加ResourceUtils工具类，支持：classpath:、classpath*:、http:、https:、ftp:、file:等资源获取

9. 优化 DateUtils，添加ConcurrentDateFormat

10. 完善优化BeanUtils，添加clone、copyConvert和copyPropertiesConvert拷贝并类型转换）

11. dream-security fixed 文件上传配置

v1.0.1 2018-05-06

1. lutool支持Csv导入(感谢`VIP`群友 @150171493)

2. lutool CaptchaUtils验证码默认改为生成Base64，方便扩展(感谢`VIP`群友 @150171493)

v1.0.0 2018-05-01

1. lutool不再内置log4j2（项目中可以根据自己的喜好修改）。

2. 修改默认的logging.pattern.console，添加行号和方法名打印（方便排插问题）。

3. lutool添加`Swagger2`自动配置。

4. lutool内置控制器`http-cache`处理，针对get请求。

5. lutool内置`ApiVersion`用于接口版本管理。

6. lutool新增`CollectionUtils,ClassUtils,ObjectUtils`。

7. lutool`BeanUtils`添加`bean diff`功能。

8. lutool`Bean-Validator`国际化文件可配置文件化。

9. dream-security国际化调整，添加国际化切换。

10. lutool-excel修复解析长数字格式和导出String类型空指针问题（感谢`pig`团队@李寻欢）。

11. lutool-excel添加`EasyCsv`，导出200万数据20秒。

12. 修复lutool-ueditor图片上传没创建目录的问题（感谢VIP群友反馈）

13. 修复dream-tpl在某些环境中乱码导致无法启动（感谢VIP群友反馈）
